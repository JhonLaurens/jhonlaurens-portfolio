Datos financieros

Migración y validación de información crítica

Contexto
Entorno financiero con información distribuida entre sistemas y necesidad de continuidad operativa.
Reto
Validar integridad, reconciliar diferencias y dejar evidencia entendible para equipos técnicos y de negocio.
Solución
Consultas SQL, reglas de calidad, bitácoras de control y revisión por etapas.
Resultado
Mayor trazabilidad del proceso y una base más clara para decisiones de migración.
  • SQL
  • PostgreSQL
  • Oracle
  • Python

Seguridad de la información

Controles, DLP y automatización de seguimiento

Contexto
Procesos de seguridad que requieren monitoreo, registro y revisión constante.
Reto
Reducir trabajo manual sin perder trazabilidad ni criterio de seguridad.
Solución
Scripts de apoyo, validaciones de integridad y consultas para revisar eventos o datos relevantes.
Resultado
Mejor visibilidad operativa y evidencia reutilizable para controles internos.
  • Python
  • SQL
  • ISO 27001
  • DLP

IA aplicada

Asistentes y clasificación con revisión humana

Contexto
Equipos con conocimiento disperso en documentos, reglas y conversaciones técnicas.
Reto
Usar IA sin perder responsabilidad sobre datos, decisiones y respuestas.
Solución
Prototipos acotados, criterios de evaluación, prompts documentados y validación humana.
Resultado
Casos más fáciles de evaluar antes de invertir en automatizaciones mayores.

Portfolio 2026

Sitio multipágina conectado a PostgreSQL

Contexto
El portfolio anterior tenía demasiado contenido, páginas de plantilla y formulario poco visible.
Reto
Construir una versión más clara, rápida y compatible con Vercel sin exponer secretos.
Solución
Vite multipágina, formulario serverless, CSP, HTML semántico y documentación de decisiones.
Resultado
Arquitectura más simple de revisar y lista para pruebas locales antes del PR.
  • Vite
  • HTML
  • CSS
  • Vercel
  • PostgreSQL
