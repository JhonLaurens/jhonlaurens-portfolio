Ingeniería Backend

Gestión Centralizada de Tarifas Bancarias

Contexto
Entorno bancario altamente regulado que requería centralizar y asegurar la gestión de tarifas de productos financieros.
Reto
Desarrollar una estructura backend acoplada bajo estándares de arquitectura limpia que permitiera auditorías automáticas de cambios.
Solución
Construcción del backend con Java Spring Boot (arquitectura hexagonal), interfaz en React y base de datos PostgreSQL.
Resultado
Cero errores en auditoría de tarifas y reducción del 90% en tiempos de aprobación.
  • Spring Boot
  • React
  • PostgreSQL
  • Arquitectura Hexagonal

Automatización y Orquestación

Sistema de Automatización e Integración de Carga Masiva

Contexto
Intercambio constante de grandes volúmenes de datos financieros con entes externos y repositorios en la nube.
Reto
Eliminar la descarga y validación manual de archivos S3/SFTP garantizando la integridad de los datos en base de datos.
Solución
Implementación de pipelines de orquestación con n8n, procesamiento de archivos con Python y almacenamiento seguro.
Resultado
Automatización completa de 15+ reportes recurrentes y erradicación de errores por intervención manual.
  • n8n
  • Python
  • S3
  • SFTP
  • PostgreSQL

Automatización Financiera

Servicio de Notificación Automatizada para Vencimientos

Contexto
Monitoreo diario de vencimientos y condiciones contractuales de portafolios financieros.
Reto
Reducir la tasa de mora y optimizar la comunicación de alertas automáticas sin exponer datos de clientes.
Solución
Programación de consultas SQL de detección en PostgreSQL, integración con n8n para generación de alertas dinámicas y plantillas HTML.
Resultado
Disminución del 12% en morosidad temprana y trazabilidad total del canal de notificaciones.
  • SQL
  • n8n
  • PostgreSQL
  • Alertas Automáticas

Infraestructura y Soporte Bancario

Soporte y Análisis de Datos sobre Core Bancario COBIS

Contexto
Entorno transaccional bancario (COBIS) que requería soporte técnico constante, corrección de inconsistencias y validación.
Reto
Identificar y resolver rápidamente problemas de consistencia en el flujo de caja diario y tablas del core bancario.

Resultado
Resolución oportuna de incidentes en bases de datos productivas (SLA > 98%) y mitigación del riesgo financiero.
