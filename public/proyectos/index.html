Backend + frontend

Spring Boot + React + arquitectura limpia

Problema
Soluciones internas financieras que requieren mantenibilidad, separación de responsabilidades y control de calidad.
Solución

Arquitectura
Dominio, casos de uso, adaptadores, infraestructura, interfaz web y control de calidad con pipeline.
Tecnologías
Java, Spring Boot, React, SQL, Bitbucket, Jenkins, SonarQube.
Impacto
Mejor mantenibilidad, control de código y sostenibilidad técnica de soluciones internas.

Portfolio

Portfolio Vite + Supabase

Problema
Portfolio anterior con exceso de información, baja claridad narrativa y formulario poco visible.
Solución
Rediseño multipágina con Vite, formulario serverless, validación de datos y persistencia en Supabase/PostgreSQL.
Arquitectura
HTML multipágina, CSS, JavaScript, API serverless, PostgreSQL/Supabase y despliegue Vercel.
Tecnologías
Vite, HTML, CSS, JavaScript, Vercel, PostgreSQL, Supabase.
Impacto
Mejor SEO, separación frontend/backend, gestión segura de variables y captura confiable de contactos.
